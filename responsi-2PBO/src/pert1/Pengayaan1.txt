Jawaban
1. Perbedaan antara Fungsi (Function) dan Prosedur (Procedure)
Dalam konsep umum pemrograman (dan khususnya diterapkan di Java melalui method), perbedaannya terletak pada nilai kembalian (return value).

 - Fungsi (Function):

    #Tujuan: Melakukan perhitungan atau memproses data dan mengembalikan hasil nilai tersebut kepada pemanggilnya.

    #Ciri di Java: Memiliki tipe data kembalian (seperti int, double, String, dll) sebelum nama method, dan wajib menggunakan kata kunci return di akhir kodenya.

    #Contoh di Kode diatas: hitungBMI() dan hitungKalori(). Method ini menghitung angka dan "mengoper" angkanya kembali untuk dipakai atau disimpan dalam variabel.

 -Prosedur (Procedure):
    #Tujuan: Melakukan serangkaian aksi atau tugas (seperti menampilkan teks ke layar, mengubah nilai variabel internal) tanpa mengembalikan nilai apapun.

    #Ciri di Java: Menggunakan kata kunci void sebagai tipe kembalian.

    #Contoh di Kode diatas: tampilInfo() dan tampilKategori(). Method ini hanya mencetak sesuatu ke konsol (System.out.println) dan tugasnya selesai di situ.

2. Pentingnya Menggunakan Konstanta (Final Variables)
Menggunakan konstanta (variabel yang nilainya tetap) sangat penting karena memberikan tiga keuntungan utama:

    a. Keamanan Data (Data Integrity): Mencegah perubahan nilai yang tidak disengaja. Nilai seperti FAKTOR_AKTIVITAS (1.2) atau PI (3.14) adalah standar baku. Jika nilai ini tidak sengaja berubah di tengah jalan karena salah ketik kode, hasil perhitungan seluruh aplikasi akan salah.

    b. Keterbacaan Kode (Readability): Mengganti "Magic Numbers" (angka misterius) dengan nama yang bermakna. Membaca rumus = berat * FAKTOR_AKTIVITAS jauh lebih mudah dimengerti daripada membaca rumus = berat * 1.2.

    c. Kemudahan Pemeliharaan (Maintainability): Jika suatu hari standar medis berubah dan faktor aktivitas menjadi 1.3, Anda hanya perlu mengganti angka tersebut di satu tempat (di deklarasi konstanta), dan seluruh perhitungan di aplikasi akan otomatis mengikuti nilai baru tersebut.

3. Mengubah Nilai Konstanta
Jika kita mencoba mengubah nilai variabel final setelah inisialisasi, program akan mengalami Compile Time Error (gagal dikompilasi).

public class UjiPengguna {
    public static void main(String[] args) {

        // 1. Mengakses nilai konstanta (INI BOLEH)
        System.out.println("Nilai awal: " + Pengguna.FAKTOR_AKTIVITAS);

        // 2. Mencoba mengubah nilai konstanta (INI DILARANG)
        // Kita mencoba mengganti 1.2 menjadi 1.5
        Pengguna.FAKTOR_AKTIVITAS = 1.5;  // <--- BARIS INI YANG AKAN ERROR

        System.out.println("Nilai baru: " + Pengguna.FAKTOR_AKTIVITAS);
    }
}

class Pengguna {
    // Definisi konstanta dengan 'final'
    public static final double FAKTOR_AKTIVITAS = 1.2;

}
4. Cara Mendefinisikan Konstanta di Java Menggunakan kata kunci final.
Biasanya dikombinasikan dengan static agar menjadi milik class (hemat memori) dan ditulis dengan huruf kapital (Snake Case).

class Pengguna {
    // -----------------------------------------------------------
    // CARA MENDEFINISIKAN KONSTANTA
    // -----------------------------------------------------------
    static final double FAKTOR_AKTIVITAS = 1.2;

    // ... method dan variabel lainnya ...
}
